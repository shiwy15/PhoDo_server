## 서비스 소개
### Phodo에서는 이런 기능을 사용할 수 있습니다
1. 유저가 업로드한 사진을 분석하여 자동 태깅 및, 자동 카테고리 분류

2. 그룹으로 실시간 동시작업과 음성통화를 통한 업무 공유

3. 회의 내용에 따른 자동 AI보고서 작성
